# TestTakify

## Description
TestTakify is a task management tool for small dev teams that embeds chat alongside tasks, eliminating context switching between communication and project tracking. Built with shadcn-svelte, it keeps discussions and decisions visible right where the work happens.

## Problems & Solutions

### Problem 1: Context Switching
Teams constantly jump between chat apps and task boards, losing focus and wasting time. TestTakify puts chat and tasks side-by-side in one view, so developers never leave their workflow.

### Problem 2: Lost Decisions
Important decisions made in chat get buried and disconnected from the tasks they relate to. TestTakify keeps conversations anchored next to the relevant work, preserving context.

### Problem 3: Unclear Task Status
It's hard to know what's actually being worked on without asking someone. Configurable task views — Kanban or List — give the whole team a clear picture at a glance.

## Key Features
- Kanban board view with drag-and-drop columns
- List view for a compact, scannable task overview
- Multi-tab views — create multiple tabs, each configurable as Kanban or List with its own settings
- Embedded chat panel alongside the task view
- Quick filters and search by assignee, label, priority
- shadcn-svelte component library for consistent, polished UI
