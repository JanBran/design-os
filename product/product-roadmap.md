# Product Roadmap

## Sections

### 1. Task Board
The core task management interface with configurable Kanban and List views, drag-and-drop, and task creation/editing.

### 2. View Tabs
Multi-tab system that lets users create, switch between, and configure multiple views with independent Kanban/List settings and filters.

### 3. Chat Panel
Embedded chat panel alongside the task view for team communication and contextual discussion about work.

### 4. Filters & Search
Quick filtering by assignee, label, and priority, plus full-text search across tasks.
